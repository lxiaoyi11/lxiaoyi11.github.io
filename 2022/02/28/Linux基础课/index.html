Linux基础课

一.常用文件命令管理

  • ctrl +c ： ( 1 ) 杀掉某个程序

    ​ ( 2 ) 取消命令，并且换

  • ctrl +u ：清空本行命令

  • tab键 ：可以补全命令和文件名

  • ls : 列出当前目录下所有文件

  • pwd ：显示当前路径

  • ls 文件名 -l ：看某一个文件的详细信息

  • ls 文件名 -hl ：人性化输出，使得输出的信息让人看起来更加的方便

  • ls -a ：显示隐藏文件（当文件名以点开头会被隐藏起来） 会显示当前目录和上层目录

  • ls -A ：显示隐藏文件，但不会显示当前目录和上层目录

  • . ：一个点表示当前目录

  • .. ：表示上层目录

  • cd ：进入某个目录

  • cd .. ：返回上层目录

  • cd - ：返回上一回呆过的目录

  • cp ：复制+粘贴+重命名 例：cp a b -r ：将a文件夹整个复制到b

  • mkdir ：创建一个文件夹

  • rm XXX ：删除普通文件

  • rm XXX -r ：删除文件夹

  • touch ：创建一个文件

  • mv ：剪切+粘贴，还有重命名

  • mv XXX YYY ：将XXX移动到YYY

  • cat ：展示文件内容

二.tmux

1 tmux的作用：

​ （1）分屏

​ （2）允许断开Terminal连接后，继续运行进程

2 tmux的结构：

​ 一个tmux可以包含多个session，一个session可以包含多个window，一个window可以包含多个pane

3 tmux的操作：

​ （1）tmux：新建一个session，其中包含一个window，window中包含一个pane，pane里打开了一个shell对话框

​ （2）先按ctrl+a，再按%：将当前pane半分成左右两个pane

​ （3）先按ctrl+a，再按“：将当前pane半分成上下两个pane

​ （4）ctrl+d：关闭当前的Pane，如果当前window的所有pane均已关闭，则自动关闭window，如果当前session下的所有window均已关闭，则自动关闭session

​ （5）先按ctrl+a，再按方向键：选择当前的pane

​ （6）鼠标拖动pane之间的分割线，可以调整分割线的位置

​ （7）按住ctrl+a的同时按方向键，可以调整分割线的位置

​ （8）先按ctrl+a，再按z：将当前的pane全屏/取消全屏

​ （9）先按ctrl+a，再按d：挂起当前的session

​ （10）tmux a ：打开之前挂起的session

​ （11）先按ctrl+a，再按s：选择其他session

  • ​ 方向键上：选择上一项，session/window/pane

  • ​ 方向键下：选择下一项，session/window/pane

  • ​ 方向键左：展开当前项，session/window

  • ​ 方向键右：闭合当前项，session/window

​ （12）先按ctrl+a，再按c：再当前的session中创建一个新的window

​ （13）先按ctrl+a，再按w：选择其他的window

​ （14）先按ctrl+a，再按PageUp：翻阅当前pane内的内容

​ （15）在tmux中选中文本时，需要按住shift键

三.vim

1.功能：

  • 命令行模式下的文本编辑器
  • 根据文件扩展名自动判别编程语言，支持代码缩进，代码高亮等功能
  • 使用方式：vim filename（如果已有该文件，则打开它；如果没有该文件，则打开一个新的文件夹，并命名为filename）

2.模式：

  • 一般命令模式：默认模式，可以复制，粘贴，删除文本
  • 编辑模式：在一般命令模式下按i，会进入编辑模式；按esc会退出编辑模式回到一般命令模式
  • 命令行模式：在一般命令模式下按：/ ？三个字母中的任意一个，会进入命令行模式，可以查找，替换，保存，退出，配置编辑器等

3.操作

  • i ：进入编辑模式

  • esc：进入一般命令模式

  • h：光标向左移动一个字符

  • j：光标向下移动一个字符

  • k：光标向上移动一个字符

  • l：光标向右移动一个字符

  • n：n表示数字，按下数字后再按空格，光标会向右移动这一行的n个字符（在一般命令模式下）

  • 0或功能键[Home]：光标移到本行开头

  • $或功能键[End]：光标移到本行末尾

  • G：光标移到最后一行

  • :n或nG：n代表数字，光标移动到第n行

  • gg：光标移动到第一行，相当于1G

  • n：n为数字，光标向下移动n行

  • /word：向光标之下寻找第一个值为word的字符串

  • ?word：向光标之上寻找第一个值为word的字符串

  • n：重复前一个查找操作

  • N：反向重复前一个查找操作

  • :n1,n2s/word1/word2/g：n1,n2为数字，在第n1行和n2行之间寻找word1这个字符串，并将该字符串替换为word2

  • :1,$s/word1/word2/g：将全文的word1替换为word2

  • :1,$s/word1/word2/gc：将全文的word1替换为word2，并且在替换前要求用户确认

  • d：删除选中的文本

  • dd：删除当前行

  • y：复制选中的文本

  • yy：复制当前行

  • p：将复制的数据在光标的下一个位置粘贴

  • u：撤销

  • ctrl+r：取消撤销

  • shift+>：将选中的文本整体向右移动

  • shift+<：将选中的文本整体向左移动

  • :w ：保存

  • :w! ：强制保存

  • :q ：退出

  • :q! ：强制退出

  • :wq ：保存并退出

  • :set nu ：显示行号

  • :set nonu ：隐藏行号

  • :set paste ：设置成粘贴模式，取消代码自动缩进

  • :set nopaste ：取消粘贴模式，开启代码自动缩进

  • gg=G：将全文代码格式化

  • :noh ：关闭查找关键词高亮

  • ctrl+q ：当vim卡死时，可以取消当前正在执行的命令

4.异常处理

每次用vim编辑文件时，会自动创建一个.filename.swp的临时文件，如果打开某文件时，该文件的swp文件已存在，则会报错，此时解决方案有两种：

  • 找到正在打开该文件的程序，并退出

  • 直接删掉该swp文件即可

三.shell语法

1.概论

shell是通过命令行与操作系统沟通的语言

shell脚本可以直接在命令行中执行，也可以将一套逻辑组织成一个文件，方便复用

Linux中常见的shell脚本有很多，常见的有：

  • Boume Shell(/usr/bin/sh或/bin/sh)
  • Boume Again Shell(/bin/bash)
  • C Shell(/usr/bin/csh)
  • K Shell(/sur/bin/ksh)
  • …

Linux系统中一般默认使用bash，所以接下来的是bash中的语法

文件开头需要写#！/bin/bash，指明bash为脚本解释器

脚本示例

新建一个test.sh文件，内容如下：

#! /bin/bash

echo “Hello World!”

运行方式

作为可执行文件（需要权限）

chmod +x test.sh //使脚本具有可执行权限

./test.sh //当前路径下执行

Hello World! //脚本输出

/home/acs/test.sh //绝对路径下执行

Hello World! //脚本输出

~/test.sh //家目录路径下执行

Hello World! //脚本输出

用解释器执行（不需要权限也可以）

bash test.sh

Hello World1 //脚本输出

单行注释

3.定义变量

定义变量

定义变量，不需要加$符号，例如：

name1=”ly” #双引号定义字符串

name2=’ly’ #单引号定义字符串

name3=ly #也可以不加引号，同样表示字符串

使用变量

使用变量，需要加上$符号，或者是${}符号。花括号是可选的，主要为了帮助解释器识别变量边界。

name=ly #定义变量为ly

echo $name #输出ly

echo ${name} #输出ly

echo ${name}beautiful #该语句相当于echo “${name}beautiful”,但由于双引号可以省略，所以等价

只读变量（相当于c++中的const）

使用readonly或者declare可以将变量变为只读

注：type命令可以检查每一个命令是什么命令

name=ly

readonly name

declare -r name

name=asd #该行命令会报错，因为此时name为只读变量

删除变量

unset可以删除变量

name=ly

unset name

echo name #会输出空命令行

变量类型

  1. 自定义变量（局部变量）->子进程不能访问的变量
  2. 环境变量（全局变量）->子进程可以访问的变量

自定义变量改成环境变量：

acs@b4a96c1c6cce:~$ name=ly

acs@b4a96c1c6cce:~$ export name #第一种方法

acs@b4a96c1c6cce:~$ declare -x name #第二种方法

环境变量改为自定义变量：

acs@b4a96c1c6cce:~$ export name=ly #定义环境变量

acs@b4a96c1c6cce:~$ declare +x name #改为自定义变量

字符串

字符串可以用单引号，也可以用双引号，也可以不用引号

单引号与双引号的区别：

  • 单引号中的内容会原样输出，不会执行，不会取变量
  • 双引号的内容可以执行，可以取变量
  • 没有引号和双引号效果一样

name=ly

echo ‘hello, ${name} \ “hh \ “ ‘ #输出hello, ${name} \ “hh \ “

echo “hello, ${name} \ “hh \ “” #输出hello ly “hh”

获取字符串长度

name=ly

echo $
